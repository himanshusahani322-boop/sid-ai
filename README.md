# 🤖 Sid AI - Smart Chatbot Website

Sid AI is a  modern and responsive AI Chatbot website ,  users questions real-time giving the answer. That project design by clean UI/UX and fast performance.

🚀 **Live Demo:** [👉 Click Here to Visit Sid AI Website]([https://sidai.netlify.app/])

---

## ✨ Features 

- **Real-time AI Chat:** Powerful Groq API ka use karke instant answers milte hain.
- **Modern UI:** Ekdum clean, attractive aur user-friendly interface.
- **Fully Responsive:** Ye website mobile, tablet aur laptop sabhi screens par perfectly chalti hai.
- **Fast & Lightweight:** Bina kisi bhaari framework ke, pure frontend tech par bani hai.

---

## 🛠️ Tech Stack 

- **HTML5:** Build Website structure.
- **CSS3:** Custom styling and responsive design.
- **JavaScript (ES6):** API integration, DOM manipulation and chatbot logic .
- **Groq API:** Fast AI processing and responses fetching.
├── index.html        # Website main frontend structure
├── style.css         # Styling and animations file
└── script.js         # API integration and chatbot logic
